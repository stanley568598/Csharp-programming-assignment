# 井字遊戲

## 介紹

這是一個簡單的井字遊戲，建構 Game tree 形成電腦的落點判斷。

從既有的走法中，歸納出遊戲的致勝之道，並將策略演繹成為程式，讓電腦與使用者對弈。

## 功能

- [x] 玩家先手 ( O )；電腦後手 ( X )
- [x] 以演算法判斷最佳走法。
